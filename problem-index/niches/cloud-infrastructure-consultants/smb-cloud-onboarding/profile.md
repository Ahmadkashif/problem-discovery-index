# SMB Cloud Onboarding

**Parent Industry:** [[industries/cloud-infrastructure-consultants|Cloud Infrastructure Consultants]]
**Category:** Low Digitized

## Profile
**Market Size:** $3B
**Share of Parent Industry:** 6%
**Digital Adoption:** Low-Medium — many SMBs still run on-prem servers or use basic hosting; consultants rely on manual assessments and phone-based discovery
**Target Buyer:** SMB Owner / IT Generalist managing 5-50 employees
**Automation Potential:** High — SMB environments are simpler and more standardized, making templated migration playbooks viable

## What Makes This a Distinct Niche
SMB cloud onboarding involves moving small businesses from on-prem file servers, Exchange, and line-of-business apps to cloud equivalents (Microsoft 365, Google Workspace, cloud-hosted LOB apps). The engagement size is small ($5K-$30K) but the volume is enormous — there are 6 million SMBs in the US with on-prem infrastructure. The consulting model is fundamentally different from enterprise: standardized playbooks, fixed-fee pricing, and minimal customization. Most cloud consultants ignore this segment because margins are thin on a per-engagement basis.

## Current Tools & Gaps
Microsoft FastTrack and Google deployment guides exist but assume technical competence the SMB buyer doesn't have. Migration tools like BitTitan MigrationWiz handle email migration but not the full workflow of assessing an SMB's environment, planning the cutover, training users, and decommissioning old hardware. The gap is an end-to-end onboarding platform purpose-built for the SMB engagement model.
