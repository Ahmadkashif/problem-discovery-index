# Design-to-Session Pipeline Manager

**Niche:** [[niches/tattoo-studios/custom-design-studios/profile|Custom Design Studios]]
**Industry:** [[industries/tattoo-studios|Tattoo Studios]]
**Type:** Build (Greenfield Opportunity)
**One-liner:** No tool exists to track a custom tattoo from initial client inquiry through reference gathering, design iterations, approval, and multi-session execution.
**Tags:** #workflow-orchestration #data-integration #ai-agent

## The Problem
Custom tattoo artists manage complex, multi-week client projects across Instagram DMs, text messages, email, and in-person consultations with no unified system. Reference images live in camera rolls, design versions are scattered across Procreate files, and session scheduling is disconnected from the design approval workflow. Artists routinely lose track of where a client is in the pipeline, leading to double-booked consultations and forgotten follow-ups.

## Why Nobody Has Built This
The tattoo industry is fragmented into independent operators with no purchasing power to attract enterprise software builders. Generic project management tools like Trello or Asana don't model the tattoo-specific workflow of reference → sketch → placement mockup → approval → session booking. Building for this market requires deep understanding of how artists actually work — which is mostly on iPads in Procreate, not in web browsers.

## What to Build
An iPad-native pipeline tool that integrates with Procreate and Instagram, allowing artists to create a client project from a DM inquiry, attach reference images, link design file versions, track approval status, and auto-generate session bookings once a design is approved. Each project shows a visual timeline from inquiry to completion with deposit status and remaining session estimates.

## Target Customer
Custom tattoo artists doing 5+ bespoke pieces per month, typically charging $150-300/hour, who are losing 3-5 hours per week to administrative overhead managing client communications across multiple channels.

## Impact If Built
Recovers 10-15 hours per month of unbilled administrative time per artist, reduces client ghosting after design phase by 30-40% through automated follow-up sequences, and increases conversion from inquiry to booked session.
