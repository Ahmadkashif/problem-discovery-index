## What Makes This a Distinct Niche
Pressure vessel and tank fabricators build equipment that operates under internal pressure — reactors, heat exchangers, storage tanks, boilers — under ASME Boiler and Pressure Vessel Code (BPVC) Section VIII. Every vessel requires: ASME code calculations (design pressure, wall thickness, nozzle reinforcement, flange ratings), a weld map showing every weld seam with its Welding Procedure Specification (WPS) and welder qualification, NDE (Non-Destructive Examination) records for radiographic, ultrasonic, or magnetic particle testing of code-required welds, and hydrostatic or pneumatic pressure test documentation. The ASME Authorized Inspector (AI) reviews all documentation before applying the ASME stamp — without which the vessel cannot legally operate. This documentation package takes 40-120 hours to compile per vessel.

## Current Tools & Gaps
Compress and PVElite provide ASME code design calculations. WeldCloud (Lincoln Electric) and Miller WeldSmart track welding parameters. Basic ERP (Epicor, JobBOSS) manages job scheduling. No platform integrates code calculations, weld map management, NDE scheduling and records, and pressure test documentation into a single vessel-centric record — the "data room" package that the ASME AI reviews before stamping.
