## What Makes This a Distinct Niche
Insurance-accepting acupuncture clinics face a billing complexity that rivals specialist medical practices but without the revenue to support dedicated billing departments. Each state has different mandates for acupuncture coverage, and within a state, each payer interprets allowed CPT codes, visit limits, and diagnosis requirements differently. A clinic in California billing Blue Shield follows entirely different rules than the same clinic billing Aetna — and both change quarterly. This creates a constant stream of denied claims, re-submissions, and revenue leakage that generic medical billing tools don't address because acupuncture-specific code combinations are edge cases in their systems.

## Current Tools & Gaps
Most clinics use general EHR/PM systems like Jane App, Unified Practice, or Jasmine. These handle scheduling and basic SOAP notes but provide no payer-specific guidance on which codes will be accepted for acupuncture, what documentation language triggers auto-denials, or when a patient's benefits have been exhausted. The result is 15-25% denial rates versus the 5-8% norm in primary care.
