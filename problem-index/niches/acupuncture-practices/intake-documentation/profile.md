# Patient Intake & Treatment Documentation

**Parent Industry:** [[industries/acupuncture-practices|Acupuncture Practices]]
**Category:** Highly Automatable

## Profile
**Market Size:** $750M (admin cost pool)
**Share of Parent Industry:** 9% (of total practice operating costs)
**Digital Adoption:** Low — Most practices use paper intake forms and manually type SOAP notes; voice-to-chart adoption is near zero.
**Target Buyer:** Practitioner-owner or office manager
**Automation Potential:** High — Intake forms are standardized, SOAP notes follow predictable patterns, and treatment documentation is highly structured.

## What Makes This a Distinct Niche
Acupuncture documentation has unique structural requirements: TCM diagnostic elements (tongue body, tongue coating, pulse qualities at three positions, TCM pattern diagnosis), point selections with laterality and needling technique, treatment modalities used (cupping, moxibustion, electroacupuncture), and outcome measures. These elements are highly structured and repetitive — an experienced acupuncturist treating low back pain uses similar point combinations and documentation patterns across dozens of patients. Yet most practitioners type or handwrite each note from scratch, spending 8-12 minutes per patient on documentation that could be 60-80% auto-generated.

## Current Tools & Gaps
Jane App and Unified Practice offer customizable SOAP note templates with basic point-selection menus. Some practitioners use Dragon Medical for dictation. No tool combines voice-or-tap clinical input with TCM-specific auto-completion, pattern-based note generation, or compliance-aware documentation that ensures all required elements are present for each payer type.
