# Teletherapy-Only Practices

**Parent Industry:** [[industries/behavioral-health-clinics|Behavioral Health Clinics]]
**Category:** Underserved Audience

## Profile
**Market Size:** $12-15B
**Share of Parent Industry:** ~12% (growing rapidly post-COVID)
**Digital Adoption:** High — digitally native by definition, but existing tools were designed for brick-and-mortar and retrofitted for telehealth
**Target Buyer:** Virtual practice owner (solo clinician scaling to 3-8 therapists) or teletherapy startup founder
**Automation Potential:** High — multi-state licensure tracking, session-by-session state verification, virtual scheduling, and consent management are all automatable

## What Makes This a Distinct Niche
Teletherapy-only practices have no physical office, which changes the entire operational model. There's no room scheduling, no walk-in capability, no physical intake paperwork — but there are unique challenges brick-and-mortar practices never face: multi-state licensure verification at every session (a therapist licensed in 5 states must confirm the patient's physical location each time), interstate compact compliance with different rules per profession (PSYPACT for psychologists, ASWB mobility for social workers), and telehealth-specific consent requirements that vary state by state. No-show rates run 15-25% higher than in-person because the barrier to canceling or ghosting is lower, and patient acquisition is entirely digital with no local physician referral network to rely on.

## Current Tools & Gaps
SimplePractice and TherapyNotes added telehealth features (integrated video, e-consent) but their core remains a brick-and-mortar EHR with telehealth bolted on. None of these platforms track multi-state licensure compliance per session, enforce location verification, handle PSYPACT/interstate compact rules, or optimize for virtual-only patient acquisition funnels. Scheduling modules assume physical rooms rather than time-zone-aware virtual capacity.
