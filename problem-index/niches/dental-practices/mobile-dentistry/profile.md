# Mobile Dentistry / Dental Vans

**Parent Industry:** [[industries/dental-practices|Dental Practices]]
**Category:** Low Digitized

## Profile
**Market Size:** $3-5B
**Share of Parent Industry:** ~2-3%
**Digital Adoption:** Low — most mobile units document on paper or tablets with offline limitations, using fixed-location PMS software retrofitted for field use
**Target Buyer:** Mobile dentistry program director, often operating within a nonprofit, FQHC, or school-district contract
**Automation Potential:** High — batch patient workflows, consent tracking, multi-payer billing, and site coordination are all structured processes currently managed manually

## What Makes This a Distinct Niche
Mobile dental units deliver care via specially equipped vans and trailers to schools, nursing homes, group homes, correctional facilities, and rural communities without fixed dental offices. The workflow is fundamentally different from a fixed practice: scheduling is site-based (coordinate with a school 3 months in advance to reserve the parking lot and gym for consent form distribution), patient flow is batched (50 children in a single school day, not individual appointments), equipment is portable with limited diagnostic capability (portable X-ray units with lower resolution, no panoramic imaging), documentation happens on tablets with unreliable cellular or Wi-Fi connectivity, and billing is a multi-payer puzzle (school-based Medicaid, facility contracts, grant funding, occasional private insurance). The patient population is predominantly underserved — low-income children, homebound elderly, incarcerated individuals — making this work mission-critical for public health but financially marginal.

## Current Tools & Gaps
Standard dental PMS (Dentrix, Open Dental) assumes a fixed location with reliable connectivity and individual-appointment scheduling. Mobile units retrofit these tools awkwardly — staff enter data in the field on tablets and sync when they return to the office, creating data lag and reconciliation errors. No PMS handles batch patient workflows (process 50 students in sequence, each with a pre-collected consent form, each needing a screening and potentially sealants or fluoride). Billing is particularly painful: a single school visit might generate 50 Medicaid claims across 3 different Medicaid managed care organizations, plus 5 private insurance claims, plus a grant-funded write-off for uninsured students. Purpose-built mobile dental software (dentistry.one, Overjet for mobile) is emerging but nascent.
