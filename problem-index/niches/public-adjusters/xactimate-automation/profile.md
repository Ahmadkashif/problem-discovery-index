# Xactimate Estimate Automation

**Parent Industry:** [[industries/public-adjusters|Public Adjusters]]
**Category:** Highly Automatable

## Profile
**Market Size:** $800M
**Share of Parent Industry:** 11%
**Digital Adoption:** Medium — All PAs use Xactimate, but estimate creation is entirely manual — line-by-line item entry, manual measurements, manual pricing selections, and manual sketch building.
**Target Buyer:** PA Estimator / Firm Owner
**Automation Potential:** High — Xactimate estimate creation follows patterns that experienced estimators recognize: specific damage types produce predictable line item sets, and measurement data can be pre-populated from aerial imagery and laser measurements.

## What Makes This a Distinct Niche
Writing Xactimate estimates is the single most time-consuming task in public adjusting — a typical residential roof and interior estimate takes 4-8 hours. Every PA must master Xactimate, but the software itself hasn't evolved to reduce the manual input burden. An experienced estimator knows that a wind-damaged roof requires a predictable set of line items (tear-off, felt, shingles, ridge cap, drip edge, pipe jacks, step flashing, waste factor, O&P), yet they must manually enter each one. This knowledge-to-keystrokes gap represents a massive automation opportunity.

## Current Tools & Gaps
Xactimate (Verisk) is the universal standard — 90%+ of property claims use it. XactAnalysis provides carrier-side estimate review. No tool automates the creation side — pre-populating estimates based on damage type, pulling measurements from aerial imagery, suggesting line items based on the scope, or generating complete estimate drafts from inspection photos and notes.
