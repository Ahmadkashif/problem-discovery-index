# Artisan Import Documentation Template System

**Niche:** [[niches/customs-brokers/indigenous-artisan-importers/profile|Indigenous & Artisan Product Importers]]
**Industry:** [[industries/customs-brokers|Customs Brokers]]
**Type:** Build (Greenfield Opportunity)
**One-liner:** No system provides artisan importers with documentation templates tailored to their unique challenges — creating customs-compliant commercial invoices from cooperative purchase records, proper product descriptions for handmade goods, and CITES compliance checks for products containing natural materials.
**Tags:** #large-language-models #compliance #automation #worker-facing

## The Problem
A fair trade importer purchasing handwoven textiles from a Mayan cooperative in Guatemala receives a handwritten receipt showing "50 huipiles, Q15,000." This is not a commercial invoice — it lacks the shipper/consignee identification, product description specificity, country of origin declaration, and per-item value breakdown that CBP requires. The importer must create a customs-compliant commercial invoice from this receipt, write product descriptions detailed enough for HTS classification ("hand-loomed cotton textile, traditional pattern, unfinished edges, approximately 24x72 inches"), and determine whether any natural dyes or materials trigger CITES review. This documentation work takes 2-4 hours per shipment and requires customs knowledge that mission-driven artisan importers typically lack.

## Why Nobody Has Built This
The artisan import market is small and fragmented — thousands of importers each doing 2-10 shipments per year, each from different suppliers with different products. No vendor has seen the revenue opportunity in building tools for a $100-200M market of cost-conscious social enterprises. The product-level variability (handmade items have no SKU or standardized description) makes template-based automation harder than for commercial goods.

## What to Build
A documentation platform for artisan importers that: (1) converts informal purchase records (handwritten receipts, cooperative production logs) into customs-compliant commercial invoices with guided data entry; (2) provides product description templates for common handmade product categories (textiles, pottery, baskets, jewelry) with the specificity CBP requires; (3) flags materials that may require CITES documentation (specific woods, animal materials, endangered plant species); (4) generates a complete import document package (invoice, packing list, country-of-origin declaration) ready for broker filing.

## Target Customer
Fair trade importers, social enterprise founders, and small retailers importing handmade goods from developing countries. Approximately 5,000-8,000 artisan importers in the US, most doing 2-10 shipments per year. Average transaction value: $100-$300 per shipment documentation service.

## Impact If Built
Reducing per-shipment documentation work from 2-4 hours to 30-45 minutes of guided template completion. Preventing CBP holds from insufficient documentation — each hold costs $500-$2,000 in port storage and delays. For mission-driven importers, every dollar saved on customs overhead is a dollar more flowing to artisan suppliers.
