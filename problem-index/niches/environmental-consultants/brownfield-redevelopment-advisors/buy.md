# Brownfield Incentive Stacking Calculator

**Niche:** [[niches/environmental-consultants/brownfield-redevelopment-advisors/profile|Brownfield Redevelopment Advisors]]
**Industry:** [[industries/environmental-consultants|Environmental Consultants]]
**Type:** Buy & Customize (Vertical Adaptation)
**One-liner:** Financial incentive calculators exist for real estate development (tax credit calculators, TIF models), but none combine brownfield-specific incentives — EPA grants, state tax credits, environmental insurance, TIF, NMTC, Opportunity Zones — into a single stacking model for contaminated site redevelopment.
**Tags:** #data-integration #automation #revenue-impact #quick-win

## The Problem
A brownfield redevelopment project typically layers 3-7 financial incentives to make the economics work: EPA Brownfields Assessment or Cleanup grants ($300K-500K), state brownfield tax credits (10-50% of cleanup costs), Tax Increment Financing (TIF), New Markets Tax Credits (NMTC), Opportunity Zone capital gains deferral, environmental insurance (cost cap policies), and HUD Community Development Block Grants. Each incentive has different eligibility criteria, application timelines, and interaction effects (some incentives reduce the basis for others). The consultant currently builds a custom Excel model for each project to estimate the combined incentive value — a 10-20 hour task that requires knowledge of each program's current rules.

## What Already Exists
Individual incentive calculators exist: NMTC investment models, Opportunity Zone benefit calculators, generic TIF projections. Real estate development pro forma tools (Argus, Excel-based models) handle standard development economics but don't model brownfield-specific incentives. EPA's Brownfields Assessment, Cleanup, and RLF grants have published award amounts but no modeling tool for probability of award or optimal grant strategy.

## The Customization Gap
The gap is a unified brownfield incentive calculator that: (1) takes site-specific inputs (location, contamination type, estimated cleanup cost, proposed end use, project economics), (2) identifies all eligible incentive programs (federal, state, and local), (3) calculates the estimated value of each incentive, (4) models interaction effects (tax credit reduces basis for depreciation, TIF captures increment that NMTC also counts), and (5) produces a stacked incentive summary showing total subsidy value and net project economics. The tool must stay current with program rule changes — EPA grant amounts, state tax credit rates, and Opportunity Zone designation status all change periodically.

## Target Customer
Brownfield consultants preparing financial feasibility analyses for developers, and developers evaluating whether a contaminated site pencils with available incentives.

## Impact If Solved
Reduces incentive analysis from 10-20 hours to 2-3 hours per project. More importantly, identifies incentive programs that consultants and developers commonly miss — particularly state and local programs with lower visibility. A missed incentive opportunity can mean $200K-1M in foregone subsidies per project.
