# Brownfield Financial Feasibility Analysis Standardization

**Niche:** [[niches/environmental-consultants/brownfield-redevelopment-advisors/profile|Brownfield Redevelopment Advisors]]
**Industry:** [[industries/environmental-consultants|Environmental Consultants]]
**Type:** Fix (Pain Point)
**One-liner:** Every brownfield feasibility analysis is a custom Excel spreadsheet with different assumptions, different cost categories, and different discount rates — making it impossible for developers and lenders to compare opportunities or for consultants to benchmark their estimates.
**Tags:** #automation #data-integration #revenue-impact #worker-facing

## The Problem
When a developer evaluates a brownfield site, the consultant produces a financial feasibility analysis comparing: acquisition cost, estimated remediation cost (with uncertainty range), available incentives, projected development cost, and expected return. Each consultant builds this analysis in their own Excel model with their own assumptions — some include environmental insurance as a cost, others don't; some discount future incentive payments, others present them at face value; some model remediation cost as a point estimate, others use Monte Carlo ranges. A developer comparing three brownfield sites evaluated by three different consultants cannot make an apples-to-apples comparison because the analyses use different frameworks.

## Why It's Still Broken
No industry standard exists for brownfield financial feasibility analysis. ASTM publishes Phase I and Phase II standards but not a financial analysis standard. EPA's Brownfields Program provides guidance on cost estimation but not on full development feasibility. Each consultant views their analytical framework as proprietary methodology. The result: brownfield financial analysis quality varies enormously, and developers make million-dollar decisions based on inconsistent inputs.

## What a Fix Looks Like
A standardized brownfield feasibility template (spreadsheet or SaaS tool) that structures the analysis into consistent categories: site acquisition, environmental investigation costs (Phase I, Phase II, supplemental), remediation costs (with low/expected/high scenarios), regulatory program costs (VCP fees, consultant fees), incentive values (itemized by program), development costs, and return metrics (IRR, cash-on-cash, equity multiple). The template enforces consistent assumptions (discount rate, contingency percentages, timeline) while allowing project-specific inputs. Consultants can customize within the framework; developers can compare across sites using consistent output metrics.

## Who Feels the Pain
Developers evaluating multiple brownfield opportunities who receive analyses in incompatible formats, and junior consultants who build feasibility models from scratch for each project because the firm has no standard template.

## Impact If Fixed
Enables developers to compare brownfield opportunities on a consistent basis for the first time. Reduces consultant feasibility analysis time from 15-25 hours to 5-8 hours by providing a structured template instead of a blank Excel workbook. Improves the quality of brownfield investment decisions by ensuring all relevant cost categories and incentives are considered.
