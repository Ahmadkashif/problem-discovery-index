# Vendor Sourcing and Coordination

**Industry:** [[event-planning|Event Planning]]
**Type:** Low Impact (Customisation Opportunity)
**One-liner:** CRM tools can track vendor contacts, but they can't match the right vendors to each event's specific style, budget, and requirements — nor manage the 200+ email threads that keep 15 vendors synchronized.
**Tags:** #gradient-boosting #decision-trees #feature-engineering #cross-validation #evaluation-metrics #expectation-variance-covariance #workflow-orchestration

## The Problem
Each event requires 10-30 vendors who must work together seamlessly on a single day. Matching the right vendors to each event involves style compatibility (a modern minimalist florist for a modern minimalist wedding), budget alignment, availability, reliability history, and interpersonal chemistry between vendor teams. A planner's vendor network is their most valuable asset — knowing that Photographer A and DJ B work well together, but Caterer C's setup always runs late. Coordinating these vendors involves hundreds of emails, countless phone calls, and a timeline that everyone must follow precisely.

## What Already Exists
The Knot/WeddingWire provide vendor directories with reviews. Aisle Planner includes vendor management tools. HoneyBook manages vendor contracts. These tools help find and track vendors but don't match vendors to events based on style/budget fit or manage the coordination workflow across 15+ vendors.

## The Customisation Gap
An intelligent vendor system needs to: (1) match vendors to events based on style, budget tier, and availability using past event data, (2) track vendor-vendor compatibility (which teams work well together), (3) automate the coordination workflow (timeline distribution, deadline reminders, day-of contact sheets), (4) manage vendor contracts and payment schedules across all vendors per event, and (5) capture post-event performance data for future matching.

## Impact If Solved
Reduces vendor sourcing time from 10-20 hours per event to 2-3 hours through intelligent matching. Eliminates coordination-related execution failures (vendor arrival times, setup conflicts) by 60-70%. Enables new planners to leverage institutional vendor knowledge rather than building their own network from scratch.
