# Pipe Sizing and Code Calculation Assistant

**Industry:** [[plumbing-contractors|Plumbing Contractors]]
**Type:** Worker Life Changing
**One-liner:** Plumbers performing new construction or addition work who need to size drain, waste, and vent systems according to UPC or IPC code tables — calculations that take 20-40 minutes with a codebook and a pencil — get a mobile calculation tool that takes fixture units and building layout as inputs and returns code-compliant pipe sizes in 2 minutes.

## The Problem
Drain, waste, and vent (DWV) pipe sizing in plumbing is governed by either the Uniform Plumbing Code (UPC) or International Plumbing Code (IPC), depending on jurisdiction, and involves looking up fixture unit values for each plumbing fixture, summing the fixture units by branch and stack, and then referencing the code tables to determine the minimum pipe diameter for each segment. For a bathroom addition with 4 fixtures, this calculation takes an experienced plumber 20-30 minutes with the codebook; for a larger multi-bathroom project, an hour or more. Junior plumbers often make sizing errors by using the wrong code edition (UPC vs. IPC) or by misreading the table headers, resulting in failed inspections and costly rework.

## Why It Matters to the Worker
Pipe sizing is a necessary but intellectually unrewarding calculation task — it requires careful reference lookup and arithmetic but no creative judgment. Experienced plumbers who have internalized the common sizing rules can do it quickly, but they still make errors when the project layout is complex or when they're working in an unfamiliar code jurisdiction. The UPC and IPC tables are not particularly accessible in their codebook format — navigating the cross-references between fixture unit tables and sizing tables is confusing even for experienced plumbers. For apprentices, pipe sizing errors are among the most common and most expensive mistakes they make on their first new construction projects.

## What a Solution Looks Like
A mobile calculator that: accepts input of each fixture type and count (toilets, lavatories, tubs, showers, kitchen sinks, etc.), building layout information (number of floors, stack configurations, building drain slope), and the jurisdiction's adopted code (UPC or IPC); performs the fixture unit summation and cross-references the appropriate sizing tables; outputs a labeled pipe sizing diagram showing minimum diameter for each branch, stack, and building drain segment. The plumber reviews the output, makes judgment calls for local conditions (longer runs may warrant upsizing), and uses the output for the installation layout.

## Impact If Solved
Reduces DWV sizing calculation time by 80-90% per job. Eliminates sizing errors caused by code table misreading — the most common source of failed plumbing rough-in inspections on new construction. Enables apprentices to perform sizing calculations correctly on their first new construction project, reducing their dependence on senior supervision for this specific task.
