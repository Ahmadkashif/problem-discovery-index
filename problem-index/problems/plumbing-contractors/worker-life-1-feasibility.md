# Feasibility — Pipe Sizing and Code Calculation Assistant

**Industry:** [[plumbing-contractors|Plumbing Contractors]]
**Problem:** [[problems/plumbing-contractors/worker-life-1|Pipe Sizing and Code Calculation Assistant]]
**Problem type:** Worker Life

**Problem summary:** Plumbers performing new construction or addition work who need to size drain, waste, and vent systems according to UPC or IPC code tables — calculations that take 20-40 minutes with a codebook and a pencil — get a mobile calculation tool that takes fixture units and building layout as inputs and returns code-compliant pipe sizes in 2 minutes.

## Solution Approach
Ship a lightweight assistant that removes repetitive steps for frontline staff, with fast feedback loops and clear handoff to humans. Deliver mobile-first jobsite workflows with photo/checklist capture and scheduling tied to field realities.

## Open Source Data
Building codes, weather, and some permit data are open; job cost, photos, and schedule data are shop-specific.

## Worth Solving?
High for workers — improves retention and quality; commercial ROI is moderate but defensible as part of the platform.
