# Feasibility — PACP/MACP Pipe Condition Assessment Coding Automation from CCTV Footage

**Industry:** [[utility-contractors|Utility Contractors]]
**Problem:** [[problems/utility-contractors/low-impact-1|PACP/MACP Pipe Condition Assessment Coding Automation from CCTV Footage]]
**Problem type:** Low Impact

**Problem summary:** NASSCO PACP/MACP coding software exists for manual defect logging, but automating the coding from CCTV footage video — identifying each defect frame, assigning the correct PACP code, and generating the condition report — requires a municipal pipe inspection-specific computer vision model that no general-purpose platform provides.

## Solution Approach
Adapt a horizontal tool with industry-specific templates, rules, and integrations so teams stop working around generic software. Deliver mobile-first jobsite workflows with photo/checklist capture and scheduling tied to field realities.

## Open Source Data
Building codes, weather, and some permit data are open; job cost, photos, and schedule data are shop-specific.

## Worth Solving?
Medium — worthwhile as a customization/efficiency lever once a clear sponsor and data owner are identified.
