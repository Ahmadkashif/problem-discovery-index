# Feasibility — Automated Shade Analysis and System Performance Modeling from Aerial Imagery

**Industry:** [[solar-installers|Solar Installers]]
**Problem:** [[problems/solar-installers/high-impact|Automated Shade Analysis and System Performance Modeling from Aerial Imagery]]
**Problem type:** High Impact

**Problem summary:** Solar designers who manually position panels in Aurora Solar to optimize around shade from trees, chimneys, and roof geometry — a design task that takes 45-90 minutes per site and requires significant experience to optimize correctly — get that layout optimization automated by an ML model that analyzes the roof's shade pattern and returns the optimal panel placement and production estimate in minutes.

## Solution Approach
Stand up a focused pilot that integrates current systems, captures the missing signals, and adds analytics/ML once the data path is stable. Deliver mobile-first jobsite workflows with photo/checklist capture and scheduling tied to field realities.

## Open Source Data
Building codes, weather, and some permit data are open; job cost, photos, and schedule data are shop-specific.

## Worth Solving?
High — core outcome/financial impact justifies investment if change management is handled.
