# Feasibility — Proactive Plan Staleness Detection and Client Re-Engagement

**Industry:** [[estate-planning|Estate Planning]]
**Problem:** [[problems/estate-planning/high-impact|Proactive Plan Staleness Detection and Client Re-Engagement]]
**Problem type:** High Impact

**Problem summary:** An estate planning firm that systematically monitors its entire client base for life events that trigger plan updates — and reaches out proactively with specific, relevant recommendations — converts a dormant client file into a sustainable revenue stream while genuinely protecting clients whose plans have drifted out of alignment.

## Solution Approach
Stand up a focused pilot that integrates current systems, captures the missing signals, and adds analytics/ML once the data path is stable. Integrate with case management, add retrieval over statutes/precedents, and ship automation with human oversight.

## Open Source Data
Public statutes and filings are open; matter notes, client data, and outcomes are proprietary, so training/grounding must use firm data.

## Worth Solving?
High — core outcome/financial impact justifies investment if change management is handled.
